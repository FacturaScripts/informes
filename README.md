# informes
El plugin Informes facilita la generación y exportación de reportes empresariales 
(ventas, clientes, stock, contabilidad, etc.).
Incluye informes predefinidos y personalizados con filtros (fechas, clientes, almacén, usuario),
exportación a PDF/CSV y opciones de programación.
Centralizando el análisis y agilizando la toma de decisiones.
- https://facturascripts.com/plugins/informes
